# 預測模型邏輯

## 輸入因子

| 因子 | 權重 | 說明 |
|------|------|------|
| FIFA 排名 | 40% | 官方世界排名 |
| 近期大賽成績 | 25% | 近兩屆世界盃/洲際盃表現 |
| 主場/地理優勢 | 20% | 北美洲主辦國與鄰近國家 |
| 賽程疲勞 | 10% | 淘汰賽加時/黃牌累積 |
| 傷病情報 | 5% | 關鍵球員缺陣 |

## 輸出

- 勝負傾向：主勝 / 和 / 客勝
- 預測比分：基於期望值估算
- 信心指數：低 / 中 / 高

## 賽後修正

每場結束後，以真實比分更新 `matches_104.json`，並計算預測命中率，寫入 `post_match_review.md`。
