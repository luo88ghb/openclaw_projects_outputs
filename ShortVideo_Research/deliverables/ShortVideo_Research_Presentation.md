# 2️⃣ 研究目標與方法
**目標**: 探索 TikTok、YouTube Shorts、Instagram Reels 三大平台的成功要素，提煉可複製公式，提供品牌與個人創作者的操作建議。\
**範圍**: 12 個具代表性的案例、平台特性、策略框架。\
**方法**: 文獻／平台官方資料 → 案例蒐集 → 案例卡建構 → 策略框架歸納 → 簡報製作。\

# 1️⃣2️⃣ 成功結構拆解
**三段式結構** (Hook‑Core‑CTA) 在全部 12 案例皆有呈現。\
- **鉤子類型**: 視覺衝擊、問題挑釁、情緒觸發、數據衝擊、音效掛鉤、人物亮相、情境設定、倒敘\
- **節奏**: 前 2‑3 秒快切、後段 4‑6 秒放慢說明、最後 2 秒 CTA。\
- **共通要素**: 9:16 垂直構圖、字幕同步、平台音樂/音效、明確 CTA。 🔥

# 1️⃣3️⃣ 可複製短影音公式
| 模板 | 結構 | 適用情境 |
|------|------|----------|
| **問題‑解決** | Hook (問句) → 痛點 (5 s) → 解決方案 (8 s) → CTA (2 s) | 產品/服務快速說明 |
| **挑戰‑挑釁** | Hook (挑戰宣言) → 示範 (8 s) → 成果對比 (5 s) → CTA (2 s) | 活動、挑戰、品牌遊戲 |
| **前後對比** | Hook (結果) → 逆向回顧 (10 s) → 結果展示 (5 s) → CTA (2 s) | 改造、變身、前後差距 |
| **教學‑步驟** | Hook (時間承諾) → 步驟 1‑3 (各 3‑4 s) → 小技巧 (2 s) → CTA (2 s) | 技能、DIY、速成教學 |
| **故事‑懸念** | Hook (懸念) → 情境鋪陳 (5 s) → 轉折高潮 (5 s) → 未完待續 CTA (2 s) | 品牌故事、情感行銷 |

# 1️⃣4️⃣ 適合品牌的操作建議
- **統一視覺**: 使用品牌色、Logo 置入、統一濾鏡 🎨
- **平台功能**: TikTok Duet、Instagram Shopping、YouTube 影片說明卡 📦
- **鉤子選擇**: 依品牌調性選擇「視覺衝擊」或「數據衝擊」\
- **投放策略**: 先測 3‑5 秒短測廣告 → 成本效益 AB 測試 → 放大投放 \
- **CTA 設計**: 商品 Tag、Link in Bio、短鏈接 \
---
# 1️⃣5️⃣ 適合個人創作者的操作建議
- **個人風格**: 保持真實、使用個人語氣 🎤
- **低成本鉤子**: 逆向敘事、挑戰宣言、日常痛點 \
- **快速製作**: 手機+CapCut，利用平台音樂庫 \
- **互動**: 問答 Sticker、Duet 挑戰、UGC 鼓勵 \
- **成長路徑**: 系列化 3‑5 支短片 → 建立期待感 📈

# 1️⃣6️⃣ 常見失敗原因
| 風險 | 描述 | 防範措施 |
|------|------|-----------|
| Hook 不夠強 | 前 2‑3 秒無視覺/情緒衝擊 | 加入震撼畫面或問題挑釁 |
| 畫面暗淡/構圖差 | 失去點擊欲望 | 提前測光線、構圖 ✔︎ |
| 版權音樂違規 | 被平台下架 | 使用平台音樂庫或免版稅音效 |
| 內容冗長 | 觀眾流失 | 控制 15‑45 秒，核心資訊 8‑12 秒內完結 |
| CTA 含糊 | 轉換率低 | 明確「點擊/購買/關注」指令 |
| 無字幕 | 無聲觀看流失 | 加入同步字幕 ✔︎ |

# 1️⃣7️⃣ 行動建議 / 下一步測試計畫
**實驗假設**: 不同 Hook 類型 (視覺衝擊 vs 問題挑釁) 會在 0‑15 s 留存上差 12% 🎯\
**行動清單**:\
1. 產出 6 支短片 (3 視覺衝擊、3 問題挑釁) – 每支 20 s。\
2. 設定 A/B 觀測指標：完播率、點讚、分享、CTA 點擊。\
3. 投放時段測試：上午 9‑11、下午 15‑17、晚上 20‑22。\
4. 使用平台分析儀表板收集數據，兩週後回顧。\
---
# 1️⃣8️⃣ 結論
- **核心洞察**: 成功短影音均遵循「3‑段式 + 強鉤子 + 平台互動功能」的通用公式。\
- **可複製要素**: 前 3 秒視覺/問題鉤子、快速剪輯、字幕同步、明確 CTA。\
- **品牌 VS 個人**: 品牌側重統一視覺與購物導流，創作者側重個人風格與低成本互動。\
- **下一步**: 依照行動建議展開 A/B 測試，持續迭代鉤子與剪輯節奏，以資料驅動提升 KPI。 🚀
