# 短影音創意題材開發專案 - 總指揮文件

## 專案概述
- **專案名稱**: 創意題材開發與評選
- **起始時間**: 2026-04-06 13:35 GMT+8
- **目標**: 創作 12 個原創影片題材，評選出 3 個最佳進行製作
- **核心要求**: 
  - 題材需有故事性、情感連結
  - 分鏡描述要具體到可直接生成參考圖
  - 影片長度依內容需求彈性調整（15-60秒）

## 團隊組成

| 角色 | Agent ID | 核心職責 |
|------|----------|----------|
| 總控PM | main | 整合創意、評選決策、最終輸出 |

## 工作流程

### Phase 1: 創意發想 (並行)
- 三位創意編劇各自產出 4 個題材
- 每個題材包含：
  - 標題與一句話摘要
  - 目標受眾
  - 影片長度建議
  - 完整分鏡腳本（含情感描述）
  - 視覺風格建議

### Phase 2: 內部評比
- 評審團依據評分標準為 12 個題材評分
- 評分維度：創意度、情感共鳴、可執行性、傳播潛力
- 選出前 3 名

### Phase 3: 最佳題材優化
- 對前 3 名題材進行細節優化
- 產出最終分鏡腳本
- 可選：生成參考速寫圖

## 評分標準 (每項 25 分，總分 100 分)

| 維度 | 評分標準 |
|------|----------|
| 創意度 | 題材是否新穎、有亮點、與眾不同 |
| 情感共鳴 | 能否觸動觀眾情緒、產生代入感 |
| 可執行性 | 製作難度是否合理、資源需求 |
| 傳播潛力 | 是否容易引起分享、討論、模仿 |
