# 2026 FIFA 世界盃進階預測系統規劃

## 目標
建立一個**模組化、可替換、不影響主系統**的進階預測引擎，為 2026 FIFA 世界盃 104 場比賽產生基於真實資料的勝負預測。

推演順序：
1. **小組賽 72 場** → 產出 32 強名單
2. **32 強 16 場** → 產出 16 強名單
3. **16 強 8 場** → 產出 8 強名單
4. **8 強 4 場** → 產出 4 強名單
5. **4 強 2 場 + 季軍戰 1 場 + 冠軍戰 1 場** → 最終冠亞季軍

**現階段只做小組賽 72 場。** 淘汰賽需等待小組賽結果產生後再進行。

---

## 設計原則

### 模組化
- 新增 `engine/advanced_predictor.py`，不修改 `worldcup_engine.py`、`scheduler.py` 等主系統檔案。
- 輸入：讀取 `data/teams.json`、`data/matches_104.json`
- 輸出：寫入 `predictions/advanced_predictions.json`
- 主系統若要使用進階預測，可選擇性載入此檔案；不使用則完全不影響。

### 資料驅動
- 所有預測必須基於可驗證的搜尋資料。
- 禁止以幻覺或單純 FIFA 排名推估。

### 可解釋
- 每場比賽的預測需附帶：資料來源、評估視角權重、最終勝率計算過程。

### 可替換
- `advanced_predictions.json` 獨立於 `matches_104.json`，可隨時移除或重新生成。

## 資料搜集規劃

### Tier 1：核心量化資料（必取）
1. **FIFA 世界排名**（現有 `teams.json` 已有，但需驗證是否為最新）
2. **球隊總身價 / 陣容估值**
3. **近期重大賽事成績**：
   - 2024 歐洲國家聯賽 / 2024 美洲盃 / 2025 亞洲盃 / 2025 非洲盃
   - 2026 世界盃資格賽戰績
   - 近期友誼賽表現
4. **最新傷病名單**（特別是關鍵球星是否缺陣）

### Tier 2：戰術相剋資料
1. **教練戰術體系**：傳控 / 防反 / 高位逼搶 / 防守反擊
2. **陣型對位**：常用陣型（4-3-3、3-4-3、4-2-3-1 等）
3. **歷史交手紀錄**：近 5~10 年對戰結果
4. **攻守風格**：進球效率、失球率、控球率、反擊效率

### Tier 3：外部變數
1. **主客場效應**：北美 3 國（美國、加拿大、墨西哥）舉辦，地理距離優勢
2. **氣候適應度**：歐洲 vs 拉丁美洲 vs 亞洲球隊的耐熱/高海拔適應
3. **博彩賠率**：主要博彩公司的勝負賠率、亞盤、大小球
4. **時區與旅行負荷**：小組賽期間移動距離

## 三視角評分模型

每場比賽給予兩隊在三個維度上的量化分數，最終合併為綜合勝率。

### Vector A：量化數據（權重 40%）
- FIFA 排名差距
- 球隊身價差距
- 近期重大賽事勝率
- 資格賽表現
- 攻防數據（進球、失球）

### Vector B：戰術相剋（權重 35%）
- 教練體系對位
- 陣型剋制
- 歷史交手
- 攻守平衡

### Vector C：外部變數（權重 25%）
- 主場/近主場優勢
- 氣候適應度
- 博彩賠率共識
- 時區/旅行負荷

### 綜合勝率計算
```
home_score = A_home * 0.4 + B_home * 0.35 + C_home * 0.25
away_score = A_away * 0.4 + B_away * 0.35 + C_away * 0.25
home_win_prob = softmax(home_score, away_score, draw_score)
```

## 小組賽推演流程

1. 為每場小組賽比賽計算三視角分數。
2. 產生勝/平/負機率。
3. 將機率轉換為最可能賽果（勝/平/負）。
4. 計算每組積分、淨勝球、進球數。
5. 決定各組前兩名。
6. 比較 12 個小組第三名，選出 8 支最佳第三名晉級。
7. 產出完整的 32 強名單與對陣樹。
8. 將 72 場預測結果寫入 `predictions/advanced_predictions.json`。

## 風險與限制

1. **資料時效性**：2026 世界盃尚未開踢，最終 26 人名單可能尚未公布，傷病名單會變動。
2. **搜尋工具限制**：某些博彩賠率可能需要登入才能取得。
3. **預測本質**：即使進階預測也無法保證 100% 準確，目標是比基礎版顯著提升。
4. **淘汰賽未知性**：小組賽結果將決定 32 強對陣，目前無法準確推演淘汰賽。
